# VR Hoops

A multiplayer VR basketball game built with Unity, Normcore, and XR Interaction Toolkit.

## Features

- Multiplayer basketball gameplay with synchronized player movement, ball physics, and game state
- Multiple play areas with exclusive player access
- Server-authoritative game state synchronization
- Real-time score and lives tracking
- Dynamic hoop positioning
- Money ball mechanics with "On Fire" state
- Player positioning and rotation on game start

## Technologies

- Unity (URP)
- Normcore (Multiplayer networking)
- XR Interaction Toolkit
- OpenXR

## Setup

1. Clone the repository
2. Open in Unity (compatible version specified in `ProjectSettings/ProjectVersion.txt`)
3. Install required packages via Package Manager
4. Configure Normcore API key in project settings
5. Build and run
